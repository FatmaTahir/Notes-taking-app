# MoodMemo

**MoodMemo** is a React-based note-taking app that lets users track their daily feelings using emojis and write notes about their day. It’s simple, colorful, and fun — perfect for journaling, mindfulness, or mood tracking.

---

## Features

- **Mood Tracking with Emojis** – Select an emoji that represents your current mood.
- **Add Notes** – Write daily notes or thoughts.
- **Delete Notes** – Remove saved notes easily.
- **Modern UI** – Built with Tailwind CSS for responsive design.
- **Timestamps** – Notes are saved with the current date.
-**Routing** – Navigate between Home and Saved Notes pages using React Router.

---

## Tech Stack

- **React** – For building UI components.
- **Tailwind CSS** – For styling the app.
- **React Router DOM** – For navigation between pages.
- **React Icons** – For mood and note icons.
